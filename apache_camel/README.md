# Apache Camel

https://camel.apache.org/

Camel is an integration framework that empowers you to quickly and easily integrate various systems consuming or producing data. It helps you to implement the ETL ([Extract-Transform-Load](https://en.wikipedia.org/wiki/Extract,_transform,_load)) pattern.
